Global Ranking API

GRAPI is free API supporting game developers. You may upload, edit and reveal your players' game result online. You could even tell the player his/her world rank.

Read More
Quick Service

Just spend 5 minutes to go through our tutorial and embedding a few lines of code into your lagacy code. Then you have connected all players of your game together.

Secured Website

Only the game developers can access and maniuplate the players game info with unique API_Key. Ordinary players could only see the top 10's names and their own play record.

Long Term Racking

The founder will keep developing and maintaining this API for a relative long time. If possible, a forum will be set up soon for anyone who needs some help.

Open Source

The API will be release under MIT lisense and anyone interested in can get the code from Github. Any contribution and folk are welcome.
